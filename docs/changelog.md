# 更新日志

所有对 XFBrowser Desktop 的重要变更都会记录在本页面中。

## [1.2.5] - 2026-04-13

### Added
- 暗色模式

### Changed
- 调整界面布局，提升操作流畅度

### Fixed
- 修复标签切换问题

## [1.2.0] - 2026-04-13

### Added
- 基于 Electron 框架实现跨平台桌面浏览器内核
- 基础网页浏览、前进后退、刷新、地址栏导航功能
- 多标签页浏览支持（目前有bug）
- 窗口最小化、最大化、关闭系统控制
- 空白页与默认主页设置

### Changed
- 优化窗口启动速度与渲染性能
- 调整界面布局，提升操作流畅度

### Fixed
- 修复窗口大小调整时界面错位问题
