Credits
=======

XFBrowser is built with the following technologies:

1. Electron
   - A framework for building cross-platform desktop applications
   - Combines Chromium and Node.js
   - Allows building applications using HTML, CSS, and JavaScript
   - Website: https://www.electronjs.org
   - License: MIT

2. Node.js
   - An open-source JavaScript runtime built on Chrome's V8 JavaScript engine
   - Enables server-side JavaScript execution
   - Provides access to the file system, networking, and OS-level operations
   - Website: https://nodejs.org
   - License: MIT

These technologies make it possible to create XFBrowser as a modern, efficient desktop application.
